  ## Usage
  In root directory enter: 
```bash
  npm i
```
To add the database to MySQL:
```bash
  mysql -u root -p
    (enter your MySQL password)
  source db/schema.sql
  use blog_db
  show databases;
  quit
```
Add your seeds:
```bash
  npm run seed
```
To start server type: 
```bash
  npm start
```
